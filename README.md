# Web Scraping Company Reviews from AmbitionBox Using Python

## Project Overview
This project demonstrates how to scrape company review data from AmbitionBox using Python. The collected data can be used for data analysis, visualization, and business insights.

## Objectives
- Extract company review data from AmbitionBox
- Store the scraped data in CSV format
- Clean and preprocess the collected data
- Perform exploratory data analysis (EDA)
- Visualize review trends

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Import required libraries
2. Send request to AmbitionBox
3. Parse HTML using BeautifulSoup
4. Extract company review details
5. Store data in a Pandas DataFrame
6. Export data to CSV
7. Perform data analysis and visualization

## Features
- Automated web scraping
- Data cleaning
- CSV export
- Review analysis
- Data visualization
